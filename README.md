# dotscripts
Contained in this repo are helpful scripts I use. 
## Installation
### Liked Installation
```
cd <.scripts/dotscripts>
make link
```
### Unlinked Installation
```
cd <.scripts/dotscripts>
make unlink
```

## Uninstallation
To uninstall the scripts run this command:
`rm ~/bin/makeflask ~/bin/makehtml`
